# **MiniMonitor**

**MiniMonitor** is a lightweight and subtle application designed to provide real-time monitoring of your system's performance metrics. It tracks CPU, GPU, and RAM usage, as well as temperature data, offering a streamlined and non-intrusive way to keep an eye on your computer’s health.

## **Features**

- **Real-Time Monitoring**: Get live updates on CPU, GPU, and RAM usage.
- **Temperature Tracking**: Monitor the temperature of your CPU and GPU to prevent overheating.
- **Subtle Display**: The app runs in the background with minimal visibility, ensuring it doesn’t disrupt your workflow.

## **Overview**

### **CPU Metrics**
- **Usage**: Displays the percentage of CPU usage.
- **Temperature**: Shows the current temperature of the CPU.

### **GPU Metrics**
- **Usage**: Displays the percentage of GPU usage.
- **Temperature**: Shows the current temperature of the GPU.

### **RAM Metrics**
- **Usage**: Displays the percentage of RAM usage.

## **How to Use**

1. **Launch the Application**: Open the application to start monitoring your system’s performance.
2. **Toggle Visibility**: Use F10 to show or hide the performance metrics overlay.  Customization of the hotkey will come later.
3. **View Metrics**: The application displays performance metrics in a sleek, minimalistic interface.
